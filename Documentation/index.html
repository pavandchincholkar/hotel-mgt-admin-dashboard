Getting Started

Follow these steps to initialize your local development cluster. If you don't have pnpm installed, run the first command to set it up globally.

// 1. Install pnpm globally (Run this if you don't have pnpm)
npm install -g pnpm

// 2. Install project dependencies
pnpm install

// 3. Launch development engine
pnpm dev

System Tip

We use pnpm because it is significantly faster and uses less disk space than standard npm. It is the professional standard for modern Next.js projects.

Global Configuration

HotelPro uses a centralized configuration pattern located at src/config/site.ts. This allows you to rebrand the entire application by modifying a single object.

export const siteConfig = {
  name: "HotelPro",
  company: "Systems",
  user: {
    name: "Alex Morgan",
    role: "General Manager"
  }
};
