# HotelPro Enterprise PMS

Thank you for purchasing HotelPro. This is a high-performance Next.js 16 Property Management System.

## 📋 Prerequisites
Before starting, ensure you have **Node.js 20+** installed. This project uses **pnpm** for faster and more reliable builds.

If you don't have pnpm installed, run:
`npm install -g pnpm`

## 🚀 Quick Start
1. `pnpm install` — (Install all project tools)
2. `pnpm dev` — (Start the development server)
3. Open `http://localhost:3000`

## 📁 Key Folders
- `/Documentation`: Detailed guide and setup instructions.
- `/src/config`: Change your brand name and logo here.
- `/src/app`: Main pages and dashboard routes.
