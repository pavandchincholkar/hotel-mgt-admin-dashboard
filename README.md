# HotelPro - Advanced Hotel Management Admin UI

HotelPro is a premium, enterprise-grade dashboard built for the 2026 industry standards. It utilizes a cutting-edge tech stack to provide an "Iron-Clad" mobile experience and a high-end desktop design.

## 🚀 Tech Stack

- Framework: Next.js 16 (App Router)
- Library: React 19
- Styling: Tailwind CSS v4
- Package Manager: pnpm
- Icons: Lucide React
- Animations: Framer Motion
- Language: TypeScript

## 📂 Project Structure & Key Files

- src/app/: Contains all application routes and pages (Dashboard, Rooms, Analytics, Reservations).
- src/components/: All UI elements. Look here for Sidebar.tsx, Header.tsx, and the DashboardLayout.tsx.
- src/config/site.ts: The central configuration file. Edit this to change the Hotel Name, User Roles,Login credentials and  Sidebar Menu items.
- src/lib/: Contains core utility functions like the cn helper for Tailwind class merging.

## 🛠️ Installation & Setup (Step-by-Step)

1. Install Node.js: Ensure you have the latest Node.js (LTS version) installed.
2. Install pnpm: Run the command: npm install -g pnpm
3. Open the Project: Extract the Zip, open VS Code, and select "Open Folder".
4. Install Dependencies: Open the VS Code terminal and run: pnpm install
5. Run Locally: To start development, run: pnpm dev
6. Build for Production: To build, run: pnpm build (then pnpm start)
